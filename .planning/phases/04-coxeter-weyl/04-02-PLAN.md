1. `reflect_phase_data(phase, g)` -- Apply group element g to a CalabiYauLite phase. Returns a new CalabiYauLite.
   - `new_kappa = np.einsum('abc,xa,yb,zc', phase.int_nums, g, g, g)` (g on Mori indices, per D-08)
   - `new_c2 = g @ phase.c2` (per D-08)
   - `g_inv = np.linalg.inv(g)` then `g_inv_int = np.round(g_inv).astype(int)` (per D-09)

   - Kahler cone: `import cytools; new_kc = cytools.Cone(rays=phase.kahler_cone.rays() @ g_inv_int)` (per D-08: Kahler rays transform by ray @ g^{-1})
   - Mori cone: `new_mori = new_kc.dual()`
   - Construct and return new CalabiYauLite with given label

2. `apply_coxeter_orbit(ekc, phases=True)` -- Main orbit expansion (per D-10 through D-14).
   
   Algorithm:
   a. Extract reflection matrices from `ekc._sym_flop_refs` (only symmetric-flop, per D-03/D-06)
   b. If empty, log and return
   c. Compute order matrix, check finite type (D-06). If infinite, log warning and return.
   d. Classify type, compute expected order (D-05)
   e. Memory estimation: `expected_order * 8 * h11^2`. Warn if > 500MB (D-07).
   f. Snapshot fundamental-domain phases and edges
   g. For each group element g from `enumerate_coxeter_group` (skip identity):
      - Compute `g_inv_int = np.round(np.linalg.inv(g)).astype(int)`
      - For each fundamental phase:
        - If `phases=True`: create reflected phase via `reflect_phase_data`, add to graph
        - Accumulate eff_cone_gens: reflected Kahler rays (phase.kahler_cone.rays() @ g_inv_int, per D-14)
      - For each fundamental edge (contraction between phase_a and phase_b):
        - Reflect the contraction curve: `g @ (sign * curve)`
        - If `phases=True`:
          - If contraction is a FLOP between two different phases: create edge between g(phase_a) and g(phase_b) (per D-12)
          - If contraction is a terminal wall (ASYMPTOTIC, CFT, SU2): create self-loop on reflected phase (per D-12)
          - If contraction is SYMMETRIC_FLOP: create self-loop on reflected phase
        - Accumulate generators regardless of phases mode (per D-13, D-14):
          - ASYMPTOTIC/CFT curves -> infinity_cone_gens
          - CFT/SU2 zero-vol divisors -> eff_cone_gens (reflect by g)
   h. Store Coxeter type info and order on ekc for the read-only API
   i. Log summary: number of reflected phases, group order, type

   **No deduplication** of reflected phases (per D-11). Each (g, fund_phase) pair is unique.

   **Label convention:** reflected phases get labels `CY_{counter}` continuing from the last fundamental-domain label number.
