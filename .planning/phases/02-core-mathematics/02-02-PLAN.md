1. `compute_gv_series(gv_invariants, curve)`:
   - Iterate n = 1, 2, 3, ... calling `gv_invariants.gv(n * curve)`
   - Stop when result is None (CYTools returns None when degree exceeds computed range)
   - Return list of int values `[GV(C), GV(2C), GV(3C), ...]`
   - If the very first call returns None, return empty list
   - Docstring cites arXiv:2303.00757 Section 2 (GV extraction)

2. `compute_gv_eff(gv_series)`:
   - `gv_eff_1 = sum(k * gv for k, gv in enumerate(gv_series, 1))`
   - `gv_eff_3 = sum(k**3 * gv for k, gv in enumerate(gv_series, 1))`
   - Return `(gv_eff_1, gv_eff_3)` as a tuple of ints
   - Raise `ValueError` if gv_series is empty
   - Docstring cites arXiv:2212.10573 below Eq. (2.7)
   - LaTeX: `n^{\mathrm{eff},p}_{\mathcal{C}} = \sum_k k^p n^0_{k[\mathcal{C}]}`

3. `is_potent(gv_series)`:
   - Return `len(gv_series) > 0 and gv_series[-1] != 0`
   - Docstring cites arXiv:2212.10573 Section 3.1 (potent curves)

4. `is_nilpotent(gv_series)`:
   - Return `len(gv_series) > 0 and gv_series[-1] == 0`
   - Docstring cites arXiv:2212.10573 Section 3.1 (nilpotent curves)
