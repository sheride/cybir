**Create tests/survey_h11_3.py (per D-06, D-07, D-08, D-09):**

The survey script:

1. Uses argparse with:
   - `--limit N` (default: None = all) — number of polytopes to process (per D-07)
   - `--max-deg N` (default: 10) — initial GV degree
   - `--outfile PATH` (default: `tests/survey_h11_3_results.jsonl`)
   - `--skip-orbit` flag — skip orbit expansion (for faster debugging runs)
   - `--start N` (default: 0) — resume from polytope index N

2. Gets all h11=3 polytopes: `from cytools import fetch_polytopes; polys = fetch_polytopes(h11=3, lattice='N')`
   This returns 244 polytopes. Filter out non-favorable: skip any where `p.is_favorable('M')` is False.
   Log which polytope ID was skipped and why.

3. For each polytope (respecting --start and --limit):
   a. Try to run the full pipeline:
      ```python
      cy = p.triangulate().get_cy()
      ekc = CYBirationalClass.from_gv(cy, max_deg=args.max_deg, verbose=False,
                                        validate_stability=True)
      ```
   b. If not --skip-orbit and ekc.sym_flop_refs is non-empty:
      ```python
      ekc.apply_coxeter_orbit(phases=True)
      ```
   c. Collect statistics per D-09:
      - `poly_id`: polytope index
      - `h11`: 3 (constant, for schema consistency)
      - `favorable`: True
      - `n_phases_fund`: number of fundamental domain phases
      - `n_phases_total`: total phases after orbit expansion (or same as fund if no orbit)
      - `contraction_types`: dict counting each ContractionType name (e.g. {"FLOP": 3, "SYMMETRIC_FLOP": 2, ...})
      - `coxeter_type`: ekc.coxeter_type (list of tuples) or None
      - `coxeter_order`: ekc.coxeter_order or None
      - `n_sym_flop_refs`: len(ekc.sym_flop_refs)
      - `n_infinity_gens`: len(ekc.infinity_cone_gens)
      - `n_eff_gens`: len(ekc.eff_cone_gens)
      - `n_unresolved`: len(ekc._unresolved_walls)
      - `time_s`: wall-clock time in seconds
      - `status`: "ok" or "error"
      - `error`: error message if status is "error"
      - `stability_validated`: True (always, since validate_stability=True)

   d. Write one JSON line per polytope to outfile (per D-06 JSONL), flush after each.
   e. Catch all exceptions per D-15: log the error, write a JSONL line with status="error",
      continue to next polytope.

4. Print summary at end: N processed, N ok, N errors, N with orbit expansion,
   aggregate contraction type distribution.

Script should be runnable:
```
conda run -n cytools python tests/survey_h11_3.py --limit 10
conda run -n cytools python tests/survey_h11_3.py  # full run
```
