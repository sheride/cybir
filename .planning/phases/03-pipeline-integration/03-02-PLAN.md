**`construct_phases(ekc, verbose=True, limit=100)`:**

The BFS loop. Key data structures:
- `known_curves: set` -- normalized curve tuples seen so far
- `curve_signs: dict[str, dict[tuple, int]]` -- `{phase_label: {curve_tuple: +1/-1}}`
- `flop_chains: dict[str, list]` -- `{phase_label: [curves flopped to reach this phase]}`
- `undiagnosed: deque` -- queue of `(wall_curve_array, source_label)` tuples
- `phase_counter: int` -- for generating labels CY_0, CY_1, ...

Initialize:
1. If verbose: `logging.basicConfig(level=logging.INFO, format="%(message)s")`
2. Get root phase, extract Mori cone generators as initial walls
3. Initialize `known_curves` from Mori cone generators (normalized)
4. Initialize `curve_signs` for root with signs of `tip @ curve` for each known curve
5. Initialize `flop_chains = {"CY_0": []}`
6. Enqueue all Mori cone generators as `(curve, "CY_0")`
7. Set `phase_counter = 1`

Loop:
```python
while undiagnosed and ekc._graph.num_phases < limit:
    wall_curve, source_label = undiagnosed.popleft()
    source = ekc._graph.get_phase(source_label)
    
    # Get GV series for this wall via flop chain
    chain = flop_chains[source_label]
    gvs_local = ekc._root_invariants.flop_gvs(chain)
    series = gvs_local.gv_series_cybir(wall_curve)
    
    # Classify
    result = classify_contraction(source.int_nums, source.c2, wall_curve, series)
    ctype = result["contraction_type"]
    
    # Build contraction
    contraction = ExtremalContraction(
        flopping_curve=np.array(normalize_curve(wall_curve)),
        contraction_type=ctype,
        gv_invariant=result.get("gv_invariant"),
        effective_gv=result.get("effective_gv"),
        zero_vol_divisor=result.get("zero_vol_divisor"),
        coxeter_reflection=result.get("coxeter_reflection"),
        gv_series=result.get("gv_series"),
        gv_eff_1=result.get("gv_eff_1"),
    )
    
    # Log (PIPE-04)
    ekc._build_log.append({
        "action": "classify",
        "source": source_label,
        "curve": normalize_curve(wall_curve),
        "type": ctype.value,
    })
    
    # Accumulate generators (D-06)
    _accumulate_generators(ekc, ctype, result)
    
    # Terminal walls
    if ctype in (ContractionType.ASYMPTOTIC, ContractionType.CFT, ContractionType.SU2):
        ekc._graph.add_contraction(contraction, source_label, source_label)
        # ^ self-loop for terminal walls, or skip edge -- check original
        logger.info("  %s: %s", ctype.display_name(), normalize_curve(wall_curve))
        continue
    
    # Symmetric flop: record but don't explore
    if ctype == ContractionType.SYMMETRIC_FLOP:
        ekc._graph.add_contraction(contraction, source_label, source_label)
        logger.info("  symmetric flop: %s", normalize_curve(wall_curve))
        continue
    
    # Generic flop: construct flopped phase
    flopped = flop_phase(source, wall_curve, series, label=f"CY_{phase_counter}")
    
    # Curve-sign deduplication
    tuple_curve = normalize_curve(wall_curve)
    if tuple_curve not in known_curves:
        known_curves.add(tuple_curve)
        _update_all_curve_signs(ekc, curve_signs, tuple_curve)
    
    flopped_tip = _compute_tip(flopped)
    flopped_signs = {c: int(np.sign(flopped_tip @ np.array(c))) for c in known_curves}
    
    existing_label = _find_matching_phase(curve_signs, flopped_signs)
    
    if existing_label is None:
        # New phase
        ekc._graph.add_phase(flopped)
        ekc._graph.add_contraction(contraction, source_label, f"CY_{phase_counter}")
        curve_signs[f"CY_{phase_counter}"] = flopped_signs
        flop_chains[f"CY_{phase_counter}"] = chain + [wall_curve]
        
        # Enqueue new phase's Mori cone walls
        flopped_gvs = ekc._root_invariants.flop_gvs(chain + [wall_curve])
        flopped_mori = flopped_gvs.cone_incl_flop()
        for gen in flopped_mori.rays():
            undiagnosed.append((gen, f"CY_{phase_counter}"))
        
        logger.info("Phase CY_%d: new (flop of %s from %s)", phase_counter, tuple_curve, source_label)
        phase_counter += 1
    else:
        # Existing phase: just add edge
        ekc._graph.add_contraction(contraction, source_label, existing_label)
        logger.info("  re-encountered phase %s via %s", existing_label, tuple_curve)

logger.info("Construction complete: %d phases, %d contractions",
            ekc._graph.num_phases, ekc._graph.num_contractions)
```
