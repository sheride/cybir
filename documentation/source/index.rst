cybir
=====

**cybir** -- GV-based extended Kahler cone construction for Calabi-Yau threefolds.

A Python package for studying the birational geometry of CY3 hypersurfaces
in toric varieties, following arXiv:2212.10573 and arXiv:2303.00757.
Built to integrate cleanly with CYTools.
